# Whx-MyMVC #

## 特点 ##
- MVC架构
- 数据库连接采用单例模式
- 数据库接口封装
- 数据库非调试模式下,具备记录错误日志功能
- 采用smarty模板引擎
- 单一入口
- 支持多模块
- 目录结构清晰
- 可配置可访问的控制器
- 简单工厂模式
- 模块公共基类单独自动加载
